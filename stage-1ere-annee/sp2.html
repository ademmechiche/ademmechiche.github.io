SP2 – Création d’une clé USB bootable

Cette situation professionnelle a été réalisée durant la première semaine de mon stage chez DEDALL.

Contexte

Un ordinateur de l’entreprise était devenu très lent et ne fonctionnait plus correctement. Une réinstallation complète de Windows était nécessaire pour le remettre en état.

Besoin

Préparer une clé USB bootable permettant d’installer Windows proprement sur le poste concerné.

Environnement

Poste Windows, connexion Internet, utilitaire de création de clé bootable, ISO Windows. Matériel simple utilisé dans l’entreprise pour les tâches internes.

Réalisation

- Téléchargement de l’ISO Windows depuis le site officiel
- Utilisation d’un outil pour créer la clé USB bootable
- Démarrage du poste sur la clé USB
- Lancement de l’installation de Windows
- Vérification du bon démarrage après installation

Bilan

Cette tâche m’a permis d’apprendre à créer un support d’installation Windows et à effectuer une réinstallation simple d’un poste de travail. J’ai compris les étapes essentielles pour remettre un ordinateur en état de fonctionnement.
